# DOM Theme Toggle Website

A simple static website created using **HTML**, **CSS**, and **JavaScript** as part of a project demonstrating **DOM manipulation** and the implementation of a **Light/Dark Mode Toggle** functionality.

## Features

* Hero section
* About section
* Why Choose Us cards
* Testimonials section
* Footer
* Light/Dark Mode Toggle
* Theme persistence using **localStorage**

## Technologies Used

* **HTML**
* **CSS**
* **JavaScript**
* **DOM Manipulation**
